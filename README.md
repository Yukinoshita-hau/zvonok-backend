# Zvonok

## Что такое Zvonok?

Zvonok — это веб-приложение для общения между пользователями с помощью текстовых сообщений и голосовой/видео связи в реальном времени. Zvonok обеспечивает возможность общаться с минимальной задержкой и низкой нагрузкой на компьютер в веб-пространстве.

## Проблема

Хотя приложений для общения по типу Discord и TeamSpeak достаточно, веб-версия Discord заблокирована в России. При использовании VPN или прокси возникает неприятная задержка, которая затрудняет полноценное общение.

## Решение

Zvonok — это российская платформа, ориентированная в первую очередь на российских пользователей (и пользователей стран СНГ), которые хотели бы использовать веб-платформу наподобие Discord, но не могут из-за высокого пинга и блокировок.

## Архитектура проекта

Zvonok состоит из двух основных компонентов:

- **Zvonok Backend** — сервер на Java/Spring Boot
- **Zvonok Frontend** — веб-приложение на React

## Zvonok MVP (v0.0.1)

### Функционал ✅

- Регистрация и авторизация пользователей
- Редактирование профиля и данных пользователя
- Система дружбы (добавление в друзья)
- Обмен текстовыми сообщениями между друзьями через WebSocket
- Групповой текстовый чат (3 и более участников) через WebSocket
- Создание приватных голосовых каналов между друзьями
- Создание и редактирование серверов с текстовыми и голосовыми каналами
- Система контроля доступа к каналам и категориям
- Приглашение пользователей на сервер через инвайт-коды
- Создание, редактирование и назначение ролей на сервере
- Функция блокировки пользователей на сервере
